Pocket Petrographer

UI Digital Microscope App Project

The Project

The goal of Pocket Petrographer was to create a digital microscope app mockup for geologists to use in the field. With an MS Geology myself, I wanted to create screens that would be simple yet thorough for hypothetical petrography applications.

User needs included:

  • Rotatable microscope view
  • Taking Photos
  • Polarization filter
  • Brightness options
  • Zoom, ruler, and notes capability
  • Ability to mark GPS location
  • Crosshair Toggle


Tools: Adobe Photoshop
